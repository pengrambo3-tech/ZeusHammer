Quick Start

Get up and running in minutes

1

Clone & Install

Download and install dependencies

git clone https://github.com/pengrambo3-tech/zueshammer.git
cd zueshammer
python3 install.py
2

Configure

Set up your API keys

# ~/.zueshammer/.env
ANTHROPIC_API_KEY=sk-ant-xxx
PERMISSION_LEVEL=semi_open
3

Run

Start the agent

# CLI mode
python3 -m src.main

# Web UI
python3 -m src.main --mode web

Python API

Integrate ZuesHammer into your projects

example.py 
from src import ZuesHammer
from src.core.config import Config

# Initialize configuration
config = Config.from_default_locations()
config.permission_level = "semi_open"
config.model = "claude-3-5-sonnet-20241022"

# Create agent instance
agent = ZuesHammer(config)

async def main():
    # Start the agent
    await agent.start()
    
    # Process user input
    response = await agent.process("Hello, help me analyze this codebase")
    print(response)
    
    # Cleanup
    await agent.stop()

# Run
main()
