⚡ ZeusHammer 核心优势

01

Local Brain 架构

最大的差异化亮点！80%的任务通过意图识别+技能匹配在本地完成，不需要调用昂贵的LLM API，大幅降低成本并提升响应速度。

02

Voice-First 设计

从语音唤醒、STT、TTS到语言自动检测，完整打造语音优先的交互体验，支持后台Daemon模式持续监听。

03

三层记忆系统

短时(LRU) + 长期(SQLite FTS5) + 工作记忆，加上向量存储(Chroma)，记忆能力最完整。

04

冥想模式 + 深度反思

业界罕见的自进化能力。AI在空闲时自动分析记忆模式、提取规则、升级技能。

05

中国LLM原生支持

内置DeepSeek/Qwen/GLM/Moonshot/ERNIE/Doubao/MiniMax等中国主流模型的统一客户端，通过chinawhapi.com一站式接入。

06

Homebrew安装

macOS用户一条命令 brew install zeushammer 即可安装，用户体验最佳。

⚠️ 已知劣势

生态规模（贡献者数量、技能市场）不及OpenClaw；消息渠道数量（13+）少于OpenClaw（30+）和Hermes（20+）；作为新兴项目，社区和文档成熟度有待验证。

🚀 一键安装

选择最适合你的安装方式

$ brew install zeushammer

🍺 Homebrew（推荐 macOS）

brew install zeushammer
zeushammer --help

📦 pipx（推荐跨平台）

pipx install zeushammer
zeushammer --help

🐳 Docker

docker pull zeushammer/zeushammer
docker run -it zeushammer/zeushammer

📜 cURL 脚本

curl -sSL https://raw.githubusercontent.com/pengrambo3-tech/ZeusHammer/master/install.sh | bash

🔧 手动安装

git clone https://github.com/pengrambo3-tech/ZeusHammer.git
cd ZeusHammer
pip install -r requirements.txt
python -m src.main --mode cli

⚙️ 配置API

# 中国大模型（推荐）
echo "OPENAI_API_KEY=your_key" >> ~/.zeushammer/.env
echo "API_BASE=https://api.chinawhapi.com/v1" >> ~/.zeushammer/.env
echo "MODEL=deepseek-chat" >> ~/.zeushammer/.env

🗑️ 卸载 ZeusHammer

# Homebrew 卸载
brew uninstall zeushammer

# pipx 卸载
pipx uninstall zeushammer

# 删除配置
rm -rf ~/.zeushammer
How to Use

使用方法

多种运行模式，满足不同场景需求

⌨️ CLI 命令行模式

# 进入交互式对话
zeushammer --mode cli
# 或
python -m src.main --mode cli

# 单次问答
zeushammer "你好，帮我写一段Python代码"

🌐 Web 界面模式

# 启动网页界面
zeushammer --mode web
# 或
python -m src.main --mode web

# 访问 http://localhost:7860

🎤 语音交互模式

# 启动语音交互
zeushammer --mode voice
# 或
python -m src.main --mode voice

# 说 "宙斯" 或 "Zeus" 唤醒

🖥️ TUI 终端界面

# 启动富文本终端界面
zeushammer --mode tui
# 或
python -m src.main --mode tui

⚙️ 安全级别设置

# 安全模式（推荐新手）
zeushammer --level safe

# 半开放模式
zeushammer --level semi_open

# 完全开放（野兽模式）
zeushammer --level full_open

🤖 配置大模型

# 编辑配置文件
nano ~/.zeushammer/config.yaml

# 可用模型：
# - deepseek-chat (推荐)
# - qwen-turbo
# - glm-4
# - claude-3-5-sonnet-20241022
# - gpt-4o
